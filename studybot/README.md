# 🤖 StudyBot — AI-Powered Study Assistant

StudyBot is a local, privacy-first AI chatbot that helps you study smarter. It combines rule-based responses, an AI language model, and PDF document Q&A into a single sleek interface.

---

## ✨ Features

| Feature | Description |
|---|---|
| **AI Chat** | Ask any academic or programming question |
| **Rule Engine** | Instant responses for study tips, greetings, help |
| **PDF Q&A** | Upload a PDF and ask questions about it |
| **Sessions** | Create, rename, switch, and delete chat sessions |
| **History** | Full conversation history persists in SQLite |
| **Streaming UI** | Typewriter effect for AI responses |
| **Markdown** | Formatted responses with code blocks, lists, headings |

---

## 📦 Installation

### Prerequisites
- Python 3.11+
- pip

### 1. Clone or download the project

```bash
cd studybot
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

> ⚠️ First-time setup downloads the SmolLM2-360M model (~700MB). This happens automatically on first run.

### 3. Run the application

```bash
python run.py
```

### 4. Open in your browser

```
http://localhost:8000
```

## 🚀 Usage

### Chat Sessions

- Click **New Session** to start a conversation
- Click any session in the sidebar to switch
- Click **✎** in the header to rename a session
- Click **✕** next to a session to delete it

### AI Chat

Ask any question:

```
What is machine learning?
Explain recursion with an example
What is the difference between a stack and a queue?
Explain Big O notation
What is Python used for?
```

### Rule-Based Shortcuts

These trigger instant predefined responses:

| Type | Examples |
|---|---|
| Greetings | `hi`, `hello`, `hey`, `good morning` |
| Help | `help`, `what can you do`, `commands` |
| Study Tips | `study tips`, `how to study` |
| Exam Tips | `exam tips`, `exam preparation` |
| Productivity | `productivity tips`, `focus tips` |
| About | `about`, `who are you` |

### PDF Upload & Q&A

1. Click **Upload PDF** in the sidebar
2. Select a `.pdf` file
3. Click **▶** to activate the PDF
4. Ask questions in the chat:
   ```
   Summarize chapter 2
   What are the key findings?
   What is the conclusion?
   Explain the methodology
   ```
5. Responses will show a **PDF** badge and cite relevant sections

## 🗄️ Database

StudyBot uses SQLite (no setup required).

**Location:** `backend/studybot.db`

**Tables:**

- `chat_sessions` — session title and timestamps
- `messages` — full message history with response type
- `uploaded_pdfs` — PDF metadata and file paths

## 🔧 API Reference

| Method | Endpoint | Description |
|---|---|---|
| POST | `/api/chat` | Send a message |
| GET | `/api/sessions` | List all sessions |
| POST | `/api/sessions` | Create a session |
| PUT | `/api/sessions/{id}` | Rename a session |
| DELETE | `/api/sessions/{id}` | Delete a session |
| GET | `/api/messages/{session_id}` | Get session messages |
| POST | `/api/upload-pdf` | Upload a PDF |
| GET | `/api/pdfs` | List uploaded PDFs |
| POST | `/api/activate-pdf/{id}` | Activate a PDF |
| DELETE | `/api/pdf/{id}` | Delete a PDF |

## 🛠️ Troubleshooting

**Model takes long to load:**
The SmolLM2-360M model is downloaded and cached automatically on first run. Subsequent starts are faster.

**PDF shows no results:**
The PDF may be image-based (scanned). OCR is not supported. Text-based PDFs work best.

**Port already in use:**
```bash
# Change port in run.py
uvicorn.run(..., port=8001)
```
